Dicoding Story is an Android apps that have many features. The application uses a Private API from Dicoding. The features are login register authentication, view stories, view the detail of story, upload new story (pictures and text) and can also to add location, view map with the position of each story by using Google Map API. I also implement animation, custom view, preferences, paging 3, unit testing and the other component. This project is one of the submissions in the “Belajar Pengembangan Aplikasi Android Intermediate” course at Dicoding Indonesia which became my first project in building an Android apps by Google's APIs from Google Cloud into the apps. I made this project from scratch until it can be used by myself and I got 4/5 stars for the submission rating.
