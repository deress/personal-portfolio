SIKAP
Project Information

Category : Website

Project Date : February 2024 - July 2024


SIKAP (Sistem Infromasi Kearsipan Penduduk) is an website application that use to data archiving management on Dinas Kependudukan dan Pencatatan Sipil Kota Malang (Disdukcapil Kota Malang), it use for the Internal Side of Disdukcapil Kota Malang. This application has many features such as login, view record of track, add archive document, view archive document, edit view document, delete view document, and many more feature that differ for each type of user. The github repository can't be accessed due to the project created is used for government instances.
