VisitCampus is an easy to use application that provides information about colleges. This application provides various information about various types of colleges such as descriptions, achievements, graduates, registration paths, and can display the college environment by using Street View from Google Maps SDK which is summarized into one complete application. This application also implements Machine Learning Model that can give insight regarding the scope of science that suit the user by answering the questions given. This information can be used by users as a guide for applying to college. There is also chatbot, it can be used by the users if they want ask something about college this chatbot can helped it. This project is my final project/capstone project in Bangkit Academy which became my first project build an Android apps as a Team with combining Machine Learning Team and Cloud Computing Team.
